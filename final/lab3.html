Lab 6: Website Evaluation — Morgan State University

Website Information

  • URL: https://www.morgan.edu
  • Name: Morgan State University
  • Target Audience: Students, faculty, prospective students, alumni

Site Organization

The site is well-structured with a navigation menu, quick links, and search. Sections are clearly divided by content: admissions, academics, campus life, and more.

CRAP Design Principles Used

  • Contrast: Good text/background color contrast.
  • Repetition: Fonts and colors are consistently applied.
  • Alignment: Headings and content are properly aligned.
  • Proximity: Related elements are grouped logically.

Accessibility Audit

Score: 78

Areas for improvement:

  • Missing alt text on some images
  • Poor contrast in footer
  • Navigation not fully keyboard accessible

Effectiveness

The site is effective for users who need to apply, explore academics, or find resources.

Efficiency

Key information is easily accessible within a few clicks.

Engagement

The design is inviting for new users but could use more interactive features like videos or hover effects.

Recommendations

Add interactive elements and improve keyboard accessibility for menus.
