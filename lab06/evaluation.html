Website Evaluation: Morgan State University

Website Information

  • URL: https://www.morgan.edu
  • Name: Morgan State University
  • Target Audience: Students, faculty, prospective students, and alumni.

Site Organization

The site is well-structured with a main navigation menu, quick links, and a search feature. Content is divided into sections such as admissions, academics, research, and campus life.

CRAP Design Principles Used

  • Contrast: The dark blue and white text color on an orange background makes the content easy to read.
  • Repetition: Consistent use of fonts and colors across the page.
  • Alignment: The text is structured with aligned headings and content blocks.
  • Proximity: Related items are grouped together, such as navigation links and related articles.

Accessibility Audit Score

Score: 78

Issues found include:

  • Missing alt text for some images
  • Lack of sufficient color contrast in footer
  • Navigation menu not keyboard accessible

Effectiveness

The site effectively provides information and supports users in completing actions and finding information like applying for admission, finding courses, and accessing student services.

Efficiency

The key information is accessible within a few clicks.

Engagement

The design is helpful for people who don't know anything about Morgan State University. It is visually appealing, but some pages could use more engaging elements such as videos.

Recommendation for Improvement

Improve the site with more interactive features instead of just text content.
